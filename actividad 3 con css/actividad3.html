Formulario de nuevo usuario









Dados los siguientes atributos: onclick, onkeydown, onmousemove.

¿Qué tienen en común? ¿En qué consiste cada uno de ellos?

Los atributos onclick, onkeydown y onmousemove son atributos de eventos en HTML que permiten ejecutar acciones en respuesta a interacciones del usuario con el navegador. Se usan comúnmente con JavaScript para crear páginas dinámicas e interactivas.

  • onclick: Se activa cuando el usuario hace clic en un elemento.
  • onkeydown: Se activa cuando el usuario presiona una tecla en el teclado.
  • onmousemove: Se activa cuando el usuario mueve el cursor del ratón sobre un elemento.
